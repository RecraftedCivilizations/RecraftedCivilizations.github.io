interface IJob
Content copied to clipboard

Represents a job consisting of a group, a name, a playerLimit, a set of DPlayer representing currentMembers, a the tasks set containing ITask, a canDemote set of other job names they can demote player from, a baseIncome, a baseXPGain, a minLvl required to join the job, if an election is required(electionRequired) and if you need a permission to join the job(permissionRequired)
